# hyperion.ng for WebOS by satgit62

[hyperrion.ng](https://github.com/hyperion-project/hyperion.ng) is an open-source ambient lighting system for TVs and audio systems that is based on real-time analysis of video and audio streams and includes a user interface for starting, stopping, and automatically launching the daemon.

✨ Features

* The webOS Loader has been optimised for use with Hyperion-webOS (PicCap), and it runs even when the NV12 option is enabled in PicCap. Unlike HyperHDR Hyperion.NG does not require any additional LUT tables.
* Native RGBW support has also been added.
* A wide range of LED controllers are supported.

✅ Requirements

   * Running HyperHDR instance in your network
   * LG webOS TV with Homebrew Channel
   * Root access may be required
